# Privacy and safety

Baby care data is sensitive family data. Treat exported JSON and chat logs as private by default.

Before publishing examples, issues, screenshots, demos, or benchmarks, remove:

- Baby names and nicknames.
- Birth dates and family identifiers.
- Exact home routines if they can identify the family.
- Real exported JSON files.
- Chat screenshots containing private information.
- API keys, bot tokens, mini-program IDs, or cloud environment IDs.

The repository ignores `data/` by default. Keep real records outside git history.
